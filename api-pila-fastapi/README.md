## Getting Started

### Requirements

-   python
-   pip
-   mongodb

### Manual Installation

```sh
# create env
cd api-pila
mkdir env
python -m venv ./env

# activate env
./env/Scripts/Activate.ps1

# Install pip dependencies
pip install -r /api/requirements.txt

# run it
python /api/main.py
```
